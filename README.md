# 게시판 만들기 미니 프로젝트

## 프로젝트 소개
스프링 부트를 사용하여 세션을 이용한 회원가입이 있고 Create, Read, Update,Delete (CRUD) 기능을 가진 게시판을 구현하는 프로젝트입니다.
